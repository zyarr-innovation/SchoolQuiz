## Building the client
        cd client
        npm install
        npm run build

## Building the server
        cd server
        npm install
        npm run build
        We can run the application using http://localhost:3000
        Note: The client pages are automatically served from this link only.

## Selecting the questions.
        mybody-complete...json contain all the questions.
        mybody-selected...json contain the selected questions from complete.
